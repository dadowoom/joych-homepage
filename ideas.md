# 기쁨의교회 홈페이지 디자인 아이디어

## 선택된 디자인 방향: 따뜻한 현대 교회 미학 (Warm Modern Sacred)

**Design Movement:** 현대적 미니멀리즘 + 따뜻한 자연 색조 (Modern Minimal + Warm Nature)

**Core Principles:**
1. 여백을 통한 신성함 표현 — 넓은 여백과 절제된 레이아웃으로 경건함을 전달
2. 따뜻한 녹색 계열 포인트 — 생명, 성장, 자연을 상징하는 깊은 녹색
3. 수직적 리듬감 — 섹션 간 명확한 구분과 흐름
4. 콘텐츠 우선 구조 — 설교/예배 정보가 즉시 눈에 들어오는 배치

**Color Philosophy:**
- 메인: 깊은 숲 녹색 (#1B5E20 계열) — 생명과 성장
- 배경: 따뜻한 오프화이트 (#FAFAF8) — 포근함
- 어두운 섹션: 짙은 남색 (#0F172A) — 신성함과 깊이
- 강조: 연한 황금빛 (#D4A853) — 영광과 빛

**Layout Paradigm:**
- 풀스크린 히어로 섹션 (소망교회 스타일)
- 8칸 퀵 메뉴 가로 배치
- 2단 콘텐츠 그리드 (영상 + 소식)
- 어두운 배경의 비전 섹션
- 4칸 관련 기관 그리드

**Signature Elements:**
1. 히어로 배경 이미지 위 반투명 오버레이 + 흰색 타이포그래피
2. 녹색 포인트 컬러의 하단 밑줄 강조 (섹션 타이틀)
3. 카드형 뉴스/설교 목록

**Typography System:**
- 헤딩: Noto Serif KR (세리프) — 전통과 권위
- 본문: Noto Sans KR (산세리프) — 가독성
- 영문 강조: Playfair Display — 우아함
