# 교회학교 페이지 분석 결과

## 소망교회 (somang.net/grow/sch) — 교회학교 메인

### 구조
- 상단 탭: "교육 1부" / "교육 2부" 탭 전환
- 각 탭 안에 부서 카드 목록 (사진 + 부서명 + 대상 연령 오버레이)
- 부서 카드: 큰 사진 배경 + 왼쪽 하단에 부서명/연령 뱃지
- 2열 그리드 레이아웃 (큰 카드 + 작은 카드 조합)
- 상단에 커리큘럼/신앙교육자료 버튼

### 부서 목록
- 교육 1부: 영아부, 유아부, 유치부, 유년부, 초등부, 소년부, 영어아동부, 어린이부
- 교육 2부: 중등부, 고등부, 드림부, 소망부, 대학부, 청년부, 청년플러스

## 한영교회 (hanyoung.org/Board/Index/5732) — 부서 상세 페이지

### 구조 (위에서 아래로)
1. 상단 서브 메뉴 탭 (영아부 | 유치부 | 유년부 | 소년부 | 청소년부 | 어와나)
2. 히어로 영역: 왼쪽 부서 사진 + 오른쪽 정보 카드 (부서명, 시간, 장소)
3. 섹션들:
   - 부서 소개 (텍스트)
   - 섬기는 이들 (부장/지도/교사 테이블)
   - 교육 목표 (번호 목록)
   - 기도제목 (번호 목록)
4. 게시판 (하단):
   - 번호 | 제목 | 작성자 | 날짜 | 조회수 컬럼
   - 페이지네이션 (1~14페이지)
   - 검색 기능 (제목/내용 선택 + 검색어 입력)

## 기쁨의교회 구현 계획

### 교회학교 메인 (/school)
- 상단 탭: "교회학교" / "청년부" 구분
- 부서 카드 그리드: 사진 + 부서명 + 대상 연령
- 부서 클릭 시 해당 부서 상세 페이지로 이동

### 부서 상세 (/school/:dept)
- 상단 부서 탭 네비게이션
- 히어로: 부서 사진 + 정보 카드
- 섹션: 부서소개, 섬기는이들, 교육목표, 기도제목
- 하단 게시판: 글 목록 + 작성 + 파일첨부 + 검색

### DB 테이블 추가 필요
- school_departments (부서 정보)
- school_posts (게시글)
- school_post_files (첨부파일)
