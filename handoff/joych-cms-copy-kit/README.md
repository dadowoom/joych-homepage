# 기쁨의교회 CMS/게시판 이식 패키지

이 패키지는 다른 교회 홈페이지 작업에서 기쁨의교회 홈페이지의 핵심 구조를 참고해 이식하기 위한 자료입니다.

포함 범위:
- 관리자 메뉴 추가/편집 구조
- 동적 페이지 렌더링 구조
- 좌측 패널형 서브페이지 레이아웃
- 공지/주보/자막/탐방/주보 광고신청 같은 게시판형 화면
- 사진 갤러리/최근행사사진 형태
- 관리자 접수 관리/성도 관리/주보 관리/팝업 관리 등 주요 CMS 패턴
- 관련 서버 라우터, DB 접근 코드, Drizzle 스키마 참고 파일

주의:
- 이 패키지는 그대로 덮어쓰기용이 아니라 이식 참고용입니다.
- 대상 프로젝트의 기존 디자인, 라우팅, 인증, DB 구조를 먼저 확인한 뒤 필요한 부분만 적용하세요.
- `.env`, 서버 비밀번호, OAuth 키, DB 비밀번호 같은 민감정보는 포함하지 않았고, 절대 코드에 넣지 마세요.
- DB 스키마가 다르면 반드시 마이그레이션을 따로 설계해야 합니다.

추천 사용법:
1. `CODEX_PROMPT.md` 전체를 다른 Codex 작업에 붙여넣습니다.
2. `source-files` 폴더의 파일들을 참고 자료로 제공합니다.
3. 대상 프로젝트에서 먼저 `git status`를 확인합니다.
4. 메뉴/CMS/게시판/관리자 기능을 한 번에 덮지 말고 영역별로 적용합니다.
5. 적용 후 타입 체크, 테스트, 빌드, 브라우저 확인을 진행합니다.
